# Handwritten Digits Recognition using Machine Learning and Deep Learning

## Project Overview

This project focuses on recognizing handwritten digits (0–9) using the MNIST dataset. Multiple Machine Learning and Deep Learning algorithms were implemented and compared to identify the most accurate classifier for handwritten digit recognition.

The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis, model development, evaluation, and performance comparison.

---

## Business Problem

Handwritten digit recognition plays a critical role in various real-world applications such as:

- Banking cheque processing
- Postal code recognition
- Automated form digitization
- Educational assessment systems
- Document processing automation

The objective is to build an intelligent classification system capable of accurately identifying handwritten digits.

---

## Dataset Information

Dataset: MNIST Handwritten Digits Dataset

- Training Images: 60,000
- Testing Images: 10,000
- Image Size: 28 × 28 Pixels
- Number of Classes: 10 (Digits 0–9)
